About AgriAId

An AI-Based Crop Disease Identification and Advisory System
Mission

To provide farmers with accurate, accessible, and affordable crop disease diagnosis using artificial intelligence and expert knowledge.

Vision

To revolutionize agricultural disease management through technology, reducing crop losses and promoting sustainable farming practices.

System Architecture

1

Image Acquisition

Farmer uploads crop leaf image

2

Crop Identification

AI identifies crop type (Corn/Rice)

3

Disease Prediction

Deep learning predicts possible diseases

4

Expert Validation

Questions refine the diagnosis

Technical Specifications
Component Technology Purpose
Backend Framework Flask (Python) Web application server
AI Framework TensorFlow/Keras Deep learning models
Database MySQL Store expert knowledge and logs
Frontend HTML5, CSS3, Bootstrap 5 User interface
Supported Crops & Diseases
Corn (Maize)
  • Common Rust
  • Gray Leaf Spot (GLS)
  • Northern Corn Leaf Blight
  • Healthy Plant
Rice
  • Rice Blast
  • Bacterial Leaf Blight
  • Brown Spot
  • Tungro Virus
  • Healthy Plant
